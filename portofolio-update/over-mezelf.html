Feiten over mezelf

HvA Student

Ik ben een CMD Student aan de Hogeschool van Amsterdam. Het is bachelorprogramma's die zich richt op interactieve digitale producten. Ik ben geboren op Curaçao en ben naar Nederland gekomen om te studeren. Ik heb CMD gekozen want ik wilde mijn creatieve vaardigheden in digitale producten inzetten. Waar ik vandaan komt is er veel weinig gefocused op digitale diensten, zoals webshops, overheids website en mobiele applicaties. Dus met de kennis van cmd wil ik bijdragen in de digitale wereld.

Hobby's

Ik ben begonnen met kunst sinds ik vijf jaar oud was. Ik hou van schilderen, tekenen, kleuren en knutselen. Toen ik ouder werd, leerde ik mezelf beter tekenen. Ik heb verschillende kunstmedium gebruikt. Denk hierbij aan softpastels, aquarelverf en alcoholstiften. In deze studiejaar ben ik begonnen met digitaal tekenen. Het is de beste beslissing die ik heb genomen. Want het is een onderdeel van mijn studie geworden. Nu dat ik een tablet heb, is het makkelijk om iconen en storyboards te illustreren.

Soort CMD-er

Ik vind dat ik meer richting visual design wil werken en een beetje front-end web development. Daardoor is het online portfolio gebouwd met CSS en HTML. Vervolgens heb ik bewust gekozen voor bepaalde ontwerp in het portfolio dat laat zien, hoe ze in visual design past. Hiervoor krijgt u een beeld over wie ik ben als CMD'er.
